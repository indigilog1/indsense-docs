    Device Configuration

    Baudrate Selection

    The device baud rate can be configured using the onboard DIP switch. The switch positions for various baud rates are listed below.

    Baudrate SW1 SW2 SW3
    9600 0 0 0
    4800 0 0 1
    19200 0 1 0
    38400 0 1 1
    57600 1 0 0
    115200 1 0 1
    128000 1 1 0
    256000 1 1 1

    Device ID Selection
