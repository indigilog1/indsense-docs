# Features & Specification

## Key Features  

- **Configurable Device Address (1~32)** – Easily set using the onboard DIP switch for flexible integration.  
- **Multi-Device Cascading** – Supports multiple modules on the **RS485 bus**, enabling efficient expansion.  
- **Wide Power Supply Range** – Operates on **5V to 24V DC**, providing versatility for various applications.  
- **Comprehensive Protection** – Features **power isolation, magnetic isolation, surge protection, lightning protection**, and a **resettable fuse** for enhanced reliability.  
- **Industrial-Grade Relays** – High-quality relays with a **10A, 250VAC contact rating**, ensuring durable and high-power switching.  
- **Configurable Baud Rate** – Adjust baud rate using the onboard switch for optimal communication speed.  
- **LED Status Indicators** – Dedicated LED indicators for each relay, showing real-time **NO (Normally Open) and NC (Normally Closed)** states for easy monitoring.  

This **industrial-grade** relay module is ideal for automation, remote control, and industrial switching applications, delivering reliability and efficiency in harsh operating conditions.
