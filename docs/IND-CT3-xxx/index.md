# Introduction

The **IND-CT3-XXX** is an **IoT-enabled 3-phase current monitoring sensor** designed for accurate and real-time measurement of electrical current in industrial and commercial environments. The device supports secure and efficient data transmission to cloud services or local servers, making it a reliable solution for **smart energy monitoring, industrial automation, and power distribution systems**.
